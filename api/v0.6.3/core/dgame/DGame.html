    Class DGame

    The main game loop manager. Meant to be overridden.

    Inherits from

    • Object (base class)

    Fields

    Name Type Description
    activeScene shared(Scene) The currently active scene
    currentState shared(EngineState) Current state of the game
    instance shared(DGame) The instance to be running from
    updateFlags shared(UpdateFlags*)

    Properties

    Name Type Description
    title [get] string Overrideable. Returns the name of the window.

    Methods

    Name Description
    run Main Game loop.
    onDraw To be overridden, called once per frame during the draw cycle.
    onInitialize To be overridden, logic for when the game is being initalized.
    onSaveState To be overridden, called when resetting and the state must be saved.
    onShutdown To be overridden, called when the came is closing.
    onUpdate To be overridden, called once per frame during the update cycle.

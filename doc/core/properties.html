    Module core.properties

    Defines template mixins for defining properties in classes.

    Templates

    Name Description
    BackedEmmittingDirtyProperty Create a property that is backed by a field, can be marked as dirty, and emits when set.
    BackedEmmittingPropertySetDirty Create a property that is backed by a field, emits when set, and marks another property as dirty when set.
    BackedPropertySetDirty Create a property that marks another property as dirty when set, and is backed by a field.

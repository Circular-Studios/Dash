    API documentation

    Module Description
    components.assets Defines the static Assets class, a static class which manages all textures, meshes, etc...
    components.camera Defines the Camera class, which controls the view matrix for the world.
    components.icomponent Defines the IComponent abstract class, which is the base for all components.
    components.mesh Defines the Mesh class, which controls all meshes loaded into the world.
    components.texture Defines the Texture class, which contains info for a texture loaded into the world.
    core.dgame Defines the DGame class, the base class for all game logic.
    core.gameobject Defines the GameObject class, to be subclassed by scripts and instantiated for static objects.
    core.gameobjectcollection Defines the GameObjectCollection class, which manages game objects and allows for batch operations on them.
    core.properties Defines template mixins for defining properties in classes.

    utility.time Defines the static Time class, which manages all game time related things.
